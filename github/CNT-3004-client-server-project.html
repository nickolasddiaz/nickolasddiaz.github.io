Socket-Based Networked File Sharing Cloud Server

A secure, multithreaded client-server file sharing system built for CNT3004 – Computer Networks
Florida Polytechnic University | Fall 2025

  • Release: Final Project
  • Forked from: SOCKET-PROGRAMMING-with-Python

Overview

This project implements a fully-featured cloud file server with both GUI and CLI clients, supporting secure file operations, real-time performance monitoring, and concurrent user connections. Built with Python socket programming, it demonstrates practical applications of network protocols, security, and distributed systems.

Key Features

Architecture & Performance

  • Multithreaded server supporting concurrent client connections
  • TLS/TCP socket encryption for secure data transmission
  • Efficient file streaming using on-the-fly ZIP compression/decompression
  • Session management with JWT-based authentication

Security

  • SQLite database for credential storage
  • Bcrypt password hashing for secure authentication
  • TLS encryption for all client-server communications

User Experience

  • Dual interface support: Full-featured GUI and command-line interface
  • Real-time progress tracking with ETA, data rates, and progress bars
  • Performance analytics with matplotlib visualizations
  • Graceful error handling with descriptive messages

Monitoring & Analytics

  • Response time histograms
  • Upload/download speed box plots
  • Statistical metrics (count, min, median, standard deviation)

Supported Operations
